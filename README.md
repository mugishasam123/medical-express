## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A text editor(preferably Visual Studio Code)

### Install

- Node js
- vs code

### Usage

#### Clone this repository

```bash
$ git clone https://github.com/mugishasam123/medical-express.git
$ cd medical-express
```

#### Run project

```bash
$ npm install
$ npm run dev
```
